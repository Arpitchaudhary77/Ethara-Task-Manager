Ethara OS - Task Management System
Overview
A full-stack project management application built for the Ethara.AI technical assessment. The app features a role-based dashboard, real-time task filtering, and a modern Material 3 interface.

Core Features
Role-Based Access Control (RBAC): * Admin: Can create, view, and delete strategic tasks.

Member: View-only access to the dashboard.

Dynamic Dashboard:

Live Search: Filter tasks by title instantly.

Priority Filtering: Toggle between High, Medium, and Low priority chips.

Deadline Tracking: Displays clear "Due before" dates for every task.

Modern UI/UX:

Dark Mode: Fully functional dark/light theme toggle.

Responsive Design: Optimized for both Web and Mobile (tested on physical Android device).

State Management: Built using StatefulWidget logic with optimized setState for high performance.

Tech Stack
Frontend: Flutter (Web/Android)

Backend: Node.js API (Google Cloud Run)

Storage: Shared Preferences (Local session handling)

Deployment: Firebase Hosting
